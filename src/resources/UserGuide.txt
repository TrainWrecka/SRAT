USER GUIDE

START UP:
Zuerst müssen Sie Ihr gewünschtes Signal laden. Dazu betätigen Sie:
1. "Load" (oben links neben den Plots)
Danach können Sie aus Ihren Dateien das gewünschte Signal laden.


Nun können Sie entscheiden, ob Sie ihr Signal im Automatischen- oder im Manuellen-Modus bearbeiten möchten:
2. entweder "Auto" oder "Manually" betätigen. (unterhalb von "Load")
Im Automatischen-Modus wählt das Tool seine Eingabewerte selber aus. 
Im Manuellen-Modus können Sie K, wp, qp, und sigma selber eingeben.

3. Die Ordnung ist immer auswählbar und befindet sich bei den Eingabewerten zuoberst.

Nun müssen Sie noch das ganze Starten. Die machen Sie indem Sie:
4. "Run" betätigen.(unten links)
So beginnt das Tool seine Berechnungen.


Ausgabe:
Über den Plots sind Tabs. Hier können Sie einzelne Plots auswählen und grösser anshauen.

Um die Plots zu vergrössern gibt es auch eine Zoomfunktion. Diese lässt sich mit Rechtem Mausklick auf den
zu vergrössernden Plot aus den Plot Einstellungen auswählen. Auch ist es möglich mit der Maus auf die zu vergrössern
gewünschte Stelle zu fahren und mit dem Mausrad zu zoomen.
Damit Sie wieder den Autoskalierung des Plots erhalten klicken Sie mit der rechten Maustaste auf den Plot und wählen
Autorange -> Both Axes oder Sie klicken (gedrückt halten) mit der linken Maustaste in den Plot und ziehen nach links.



Zusatz Settings:
-Das Tool besitzt ein "Settings"- Fenster. Dieses können Sie in der Task-Leiste am oberen Rand öffnen.
 Hier können Sie verschiedenste Einstellungen vornehmen.
 Das Tool kann das Rauschen der Signale filtern und diese gefiltert anzeigen.
 Sie können aber auch die ungefilterten Signale anzeigen lassen.
 Ihre Eingaben können Sie mit den Knöpfen unten bestätigen/wiederrufen.
